Introducción

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Fue desarrollado por Tim Berners-Lee, un científico británico, en 1991 mientras trabajaba en el CERN (Organización Europea para la Investigación Nuclear). Berners-Lee diseñó HTML como una forma de compartir documentos científicos y conectar información a través de hipervínculos.

HTML permite a los desarrolladores:

  • Definir la estructura y el contenido de una página web.
  • Utilizar etiquetas que describen elementos como encabezados, párrafos, enlaces e imágenes.
  • Incluir otros componentes multimedia.
  • Mejorar la semántica, accesibilidad y capacidades multimedia con HTML5.

Elementos Básicos

En HTML, los elementos son las etiquetas utilizadas para estructurar el contenido de una página. Los elementos básicos incluyen:

  • html: Define el documento HTML.
  • head: Contiene información meta sobre el documento, como el título y los enlaces a hojas de estilo y scripts.
  • title: Define el título del documento, que se muestra en la barra de título del navegador.
  • body: Contiene el contenido visible de la página, como texto, imágenes y enlaces.
  • h1, h2, h3, h4, h5, h6: Definen los encabezados o títulos de diferentes niveles.

Para mas información acerca de los distintos elementos disponibles en HTML puede acceder aquí.

Etiquetas y Atributos

Cada elemento en HTML puede tener atributos que modifican su comportamiento o apariencia. Los atributos se definen dentro de la etiqueta de inicio y proporcionan información adicional. Algunos ejemplos comunes de atributos son:

  • href: Utilizado en enlaces (a) para especificar la URL de destino.
  • src: Utilizado en imágenes (img) para especificar la ruta de la imagen.
  • alt: Proporciona un texto alternativo para imágenes que no se pueden mostrar.
  • class: Asigna una clase CSS a un elemento para aplicar estilos específicos.

Enlaces e Imágenes

HTML proporciona elementos para agregar enlaces e imágenes a una página web.

Los enlaces son elementos fundamentales en HTML que permiten conectar diferentes páginas web y recursos en la World Wide Web. Se crean utilizando la etiqueta a (ancla) y se definen mediante el atributo href, que especifica la URL de destino. Cuando los usuarios hacen clic en un enlace, son redirigidos a la dirección web asociada.

<a href="https://www.ejemplo.com" target="_blank">Abrir en nueva pestaña</a>

El ejemplo de arriba nos muesta el código de un enlace a la página "https://www.ejemplo.com", pero tambien hemos utilizado otro atributo llamado target que determina donde se abriran los enlaces. Este puede tener los siguientes valores:

  • _self (predeterminado): El enlace se abre en la misma ventana o pestaña.
  • _blank: El enlace se abre en una nueva ventana o pestaña del navegador.
  • _parent: El enlace se abre en el marco superior del marco actual.
  • _top: El enlace se abre en la ventana completa del navegador, reemplazando cualquier marco existente.

Listas y Tablas

HTML ofrece elementos para crear listas y tablas que ayudan a organizar el contenido de manera estructurada.

Para listas, los elementos principales son:

  • ul: Lista no ordenada.
  • ol: Lista ordenada.
  • li: Elemento de la lista.

Para tablas, los elementos principales son:

  • table: Define una tabla.
  • tr: Define una fila en la tabla.
  • td: Define una celda en la fila.
